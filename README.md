# MAGIC HAIR by Natali – interaktive Salon-Webseite

Dies ist die erste eigenständige Konzeptversion für MAGIC HAIR by Natali. Kontaktdaten,
Bewertungen, Zertifikate und Social-Media-Vorschauen sind noch Platzhalter.

## Vorschau öffnen

Die Datei `index.html` kann direkt im Browser geöffnet werden. Für die beste Vorschau kann
der Ordner alternativ über einen kleinen lokalen Webserver bereitgestellt werden.

## Bereits umgesetzt

- scrollgesteuerte Salon-Reise im Hero als fünfteilige Bildsequenz
- vorbereiteter Platz für das spätere echte Hero-Video
- Profilbereich der Hauptfriseurin
- filterbare Galerie mit vergrößerbarer Bildansicht
- scrollgesteuerte Zertifikate auf einer Kreisbahn, mobil als Wischgalerie
- manuell steuerbare Testimonials
- Social-Media-Karten mit fiktiven Vorschauen
- validiertes Demo-Kontaktformular ohne tatsächlichen Versand
- Mobilansicht, Tastaturbedienung und reduzierte Bewegung
